The trick to get Free Data with Ooredoo

July 6th, 2026 — Abdelali221

A theory on how this is even possible

What is the trick, exactly?

Ooredoo users noticed this weird thing where when they turn Airplane Mode on and off they get internet access for a second even though they have no data on their account (you might want to try it yourself if you happen to be an Ooredoo client).
This trick has been shared oraly between users and is getting spread little by little (I personally discovered it by accident). This trick can be used to have phone calls using VoIP apps like WhatsApp or Facebook Messenger for basically free, wiping the need to buy an internet plan like you would if this didn't exist, or even watch YouTube by doing the trick every time it buffers to get 5 to 20 seconds of video.

The theory behind this

My first theory is that when you connect to the network, you get full internet access for a fraction of seconds, and just after that the client manager server checks whether or not you have an ongoing plan or have some data in your account, THEN when it figures out that you don't it puts a firewall in your connection only allowing communication to services like choof.ooredoo.dz.
The slowness of this process allows you to have enough time to transceive with the internet (i.e watch YouTube or accessing a website).

Another explanation would be that there's a bug in the client manager that gives the user about 200 to 400KB of data (which is the value I get most times, though you can get up to ~20MB with a Sahla Box SIM card), which isn't noticeable most times until you give it attention.

The weird thing about this is that it only affects Ooredoo, which is weird since you might think that they most likely all use the same software provided by manufacturers like Huawei or Nokia (these are the main manufacturer of cellular coverage and management systems).
