Useful Tools :

I've made some tools that can be quite handy, such as :

SaveMiiRduino

SaveMiiRduino is a SaveMii module alternative,
it uses the SaveMiiFrii method to put the Wii in its recovery mode.
This tool can, in theory, used as an unbrick tool.

Disclaimer : This tool was never tested, it means that I can't guarantee any result.

Links :

Github Repository

Arduino 2 GC Controller

This tool can emulates a GC Controller using a 16MHz Arduino Board.
